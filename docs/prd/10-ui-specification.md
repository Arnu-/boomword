# UI规范与交互说明

## 1. 设计原则

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| 游戏化 | 界面充满趣味性，使用动效和反馈增强游戏感 |
| 简洁 | 信息层级清晰，减少用户认知负担 |
| 一致性 | 统一的视觉语言和交互模式 |
| 可访问性 | 考虑不同用户的使用需求 |
| 响应式 | 适配多种屏幕尺寸 |

### 1.2 设计关键词

- 活泼、轻松、有趣
- 学习与娱乐结合
- 成就感和正反馈
- 清新、明亮的色调

#### 9.2.3 顶部导航栏 (Header)

#### 9.2.5 中间主内容区

##### A. 英雄区域 (Hero Banner)

- **背景**: 深色渐变卡片，可能带有微妙的泡泡装饰动画
- **内容**:
  - 顶部: 泡泡动画图标（浮动的透明泡泡）

  - 副标题: "戳破泡泡，输入单词，赢取高分！"（浅灰色说明文字）

    - "▶ 立即开始" — 主要按钮（紫蓝色渐变背景，白色文字）
    - "↔ 练习模式" — 次要按钮（深色背景带边框，白色文字）
