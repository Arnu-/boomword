## 1. 产品概述

### 1.1 产品定位

BoomWord 是一款寓教于乐的单词学习与键盘练习游戏，通过打字击破单词泡泡的方式，帮助用户在游戏中掌握英语词汇，同时提升键盘打字技能。

### 1.2 目标用户

| 用户群体 | 核心需求 | 使用场景 |
|----------|----------|----------|
| 中小学生 | 配合教材单元词汇学习 | 课后复习、考前突击 |
| 大学生 | 四六级、考研词汇备考 | 备考期间强化记忆 |
| 英语学习爱好者 | 扩展词汇量、保持语感 | 碎片时间学习 |
| 键盘打字初学者 | 提升打字速度和准确性 | 娱乐化练习 |

### 1.3 核心价值主张

1. **游戏化学习** - 将枯燥的单词记忆转化为有趣的泡泡消除游戏
2. **双重提升** - 同时练习英语词汇和键盘打字技能
3. **循序渐进** - 科学分级，从易到难逐步提升
4. **社交激励** - 排行榜和成就系统激发学习动力

### 1.4 产品愿景

成为最受欢迎的游戏化英语单词学习工具，让每一个用户都能在快乐中掌握英语词汇。

## 7. 安全要求

### 7.1 数据安全

| 安全项 | 要求 |
|--------|------|
| 密码存储 | bcrypt 加密，不存储明文 |
| 敏感数据 | AES-256 加密存储 |
| 数据传输 | 全站 HTTPS |
| SQL注入 | 参数化查询防护 |
| XSS攻击 | 输入输出过滤 |

### 7.2 认证授权

| 安全项 | 要求 |
|--------|------|
| 认证方式 | JWT Token |
| Token有效期 | 7天，支持续签 |
| 权限控制 | RBAC 角色权限模型 |
| 登录限制 | 失败5次锁定30分钟 |

### 7.3 接口安全

| 安全项 | 要求 |
|--------|------|
| 请求签名 | 关键接口需签名验证 |
| 频率限制 | 单IP 100次/分钟 |
| 防重放 | 时间戳+随机数 |

## 8. 术语表

| 术语 | 英文 | 说明 |
|------|------|------|
| 词库 | Word Bank | 一组按主题/教材组织的单词集合 |
| 关卡 | Chapter | 词库下的大单元，包含多个小节 |
| 小节 | Section | 关卡下的最小游戏单位，通常10-15词 |
| 连击 | Combo | 连续正确输入的次数 |
| 掌握度 | Mastery | 用户对单词的熟悉程度 |
| 泡泡 | Bubble | 游戏中显示单词的视觉元素 |
| 经验值 | EXP | 用户通过学习获得的成长点数 |
| 评星 | Star Rating | 关卡完成后的评价等级（1-3星） |
