# BoomWord - 单词泡泡消消乐 需求说明文档

## 1. 产品概述

### 1.1 产品定位
BoomWord 是一款寓教于乐的单词学习与键盘练习游戏，通过打字击破单词泡泡的方式，帮助用户在游戏中掌握英语词汇，同时提升键盘打字技能。

### 1.2 目标用户
- 中小学生（配合教材单元词汇学习）
- 大学生（四六级、考研词汇备考）
- 英语学习爱好者
- 键盘打字初学者

### 1.3 核心价值
- 游戏化学习，提升学习兴趣
- 结合键盘练习，一举两得
- 分级分类，循序渐进
- 社交竞争，激励学习

## 4. 关卡体系

### 4.1 关卡结构

```
词库 (如：四级核心词)
├── 关卡 1 (Chapter 1) - 30个词
│   ├── 第1节 (1-1) - 10个词
│   ├── 第2节 (1-2) - 10个词
│   └── 第3节 (1-3) - 10个词
├── 关卡 2 (Chapter 2) - 30个词
│   ├── 第1节 (2-1)
│   ├── 第2节 (2-2)
│   └── 第3节 (2-3)
└── ...
```

### 4.2 关卡模式

#### 4.2.1 练习模式 (Practice Mode)
- **泡泡显示**：同时显示英文和中文
- **时间限制**：无时间限制 / 宽松时间
- **目的**：熟悉单词拼写，建立记忆
- **可重复次数**：无限制

#### 4.2.2 挑战模式 (Challenge Mode)
- **泡泡显示**：仅显示中文
- **时间限制**：有时间限制
- **解锁条件**：练习模式至少1星通关
- **目的**：检验学习成果
- **奖励**：更高分数、更多经验值

#### 4.2.3 速度挑战 (Speed Challenge)
- **泡泡显示**：仅显示英文
- **玩法**：看英文，快速输入，考验反应和打字速度
- **解锁条件**：挑战模式2星以上通关
- **排行榜**：独立速度排行榜

### 4.3 解锁机制

```
关卡解锁规则：
1. 第一关第一节 (1-1) 默认解锁
2. 完成当前小节（任意星级）→ 解锁下一小节
3. 完成关卡所有小节 → 解锁下一关卡
4. 挑战模式需练习模式1星以上解锁
5. 速度挑战需挑战模式2星以上解锁
```

### 4.4 评星机制

#### 4.4.1 练习模式评星
| 星级 | 条件 |
|------|------|
| 1 星 | 正确率 ≥ 60% |
| 2 星 | 正确率 ≥ 80% |
| 3 星 | 正确率 = 100% |

#### 4.4.2 挑战模式评星
| 星级 | 条件 |
|------|------|
| 1 星 | 正确率 ≥ 60% 且 剩余时间 > 0 |
| 2 星 | 正确率 ≥ 80% 且 剩余时间 ≥ 30% |
| 3 星 | 正确率 = 100% 且 剩余时间 ≥ 50% |

## 5. 游戏玩法

### 5.1 主游戏流程

```
开始游戏
    ↓
显示倒计时 (3, 2, 1, GO!)
    ↓
泡泡从屏幕边缘飘入
    ↓
玩家输入单词 + 回车
    ↓
正确 → 泡泡爆炸 + 得分 + 音效
错误 → 泡泡抖动 + 扣分/不扣分 + 音效
    ↓
所有泡泡清除 或 时间耗尽
    ↓
结算画面（得分、用时、正确率、星级）
```

### 5.2 泡泡机制

#### 5.2.1 泡泡属性
- **大小**：根据单词难度，难度越高泡泡越大
- **颜色**：不同难度不同颜色
  - 1星：绿色
  - 2星：蓝色
  - 3星：黄色
  - 4星：橙色
  - 5星：红色
- **移动**：缓慢漂浮移动，可设置移动速度
- **数量**：屏幕同时显示 5-10 个泡泡

#### 5.2.2 泡泡显示内容
| 模式 | 显示内容 |
|------|----------|
| 练习模式 | 英文（大）+ 中文（小）|
| 挑战模式 | 仅中文 |
| 速度挑战 | 仅英文 |

### 5.3 输入机制

#### 5.3.1 输入方式
- 键盘输入英文字母
- 按回车键确认提交
- 按 Backspace 删除
- 按 ESC 清空当前输入

#### 5.3.2 输入反馈
- **输入区域**：屏幕底部显示当前输入内容
- **实时匹配**：输入时高亮匹配的泡泡（可选开关）
- **错误提示**：输入错误时震动/变红提示

### 5.4 计分规则

#### 5.4.1 基础得分
```
基础分 = 单词难度分值
时间奖励 = 基础分 × (剩余时间比例 × 0.5)
连击奖励 = 基础分 × (连击数 × 0.1)，最高 ×2

总分 = 基础分 + 时间奖励 + 连击奖励
```

#### 5.4.2 连击机制
- 连续正确输入累积连击
- 连击数显示在屏幕上
- 输入错误连击归零
- 连击里程碑动画（10连、20连、50连...）

#### 5.4.3 扣分机制（可配置）
- 练习模式：不扣分
- 挑战模式：每次错误扣除 5 分
- 速度挑战：每次错误扣除 10 分

### 5.5 计时机制

| 模式 | 时间设置 |
|------|----------|
| 练习模式 | 无限制 / 可选 5 分钟 |
| 挑战模式 | 每词 10 秒，10词 = 100秒 |
| 速度挑战 | 每词 5 秒，10词 = 50秒 |

```
└─────────────────────────────────────────────────────┘
```
